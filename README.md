# ASGP Interactive Story — Cinematic React Animation

An interactive, cinematic single-page React application telling the complete story of how the **AgenticSearchGatewayProtocol** (ASGP) works — from a real user's problem to a final agentic answer.

## Quickstart

```bash
npm i
npm run dev
# Open http://localhost:5173
```

## Controls

| Key | Action |
| --- | ------ |
| `Space` | Play / Pause |
| `←` `→` | Previous / Next step |
| `R` | Restart |

## Story Arc

- **Act 1-2** — Alex's scattered data world & the chaos of direct connections
- **Act 3** — Configuring ASGP (YAML + security)
- **Act 4** — System boot sequence
- **Acts 5-7** — The full request lifecycle (tokenization, parallel execution, retries, LLM summarization)
- **Final** — Unified dashboard with stats

## Tech Stack

React 18 · TypeScript · Vite · Framer Motion · React Flow · Zustand · Tailwind CSS
